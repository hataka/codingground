## やさしいAndroidプログラミング 高橋麻奈 第３版 SB Creative
### Sample082 検索画面を起動する p.215
#### 解説
　アクションとしてACTION_SEARCHを渡しています。putExtra()メソッドを使うと。名前と値の組み合わせをデータとして渡すことができます。ここで検索データとして「Android」を渡すことで、キーワード「Android」の検索を行うことにします。  
　「検索」ボタンを押すと、ここでは複数の検索７アプリが表示されました。  
　ACTION_SEARCHを受け取るアプリが複数存在するわけです。  
　そこで実行画面では、「Browser」(Web ブラウザ)を選択しています。
