# Lesson8 アプリの連携 コンテントプロバイダ「を使う
## Sample086 ギャラリーの画像を取得する p.230
コンテントプロバイダからデータを主創するには、URLを指定してカーソルでデータの集合を読み込んだり、コンテントリゾルバ(ContentResolver)クラスのメソッドで内容を読み込めるようにします。  
前者の方はSample5で実践したので、今度は後者の方法でデータを取り出してみます。標準アプリであるギャラリ(フォト)を起動した上で写真を選択させ、そのURIをもとにコンテントプロバイダーから画像を取り込みます。
